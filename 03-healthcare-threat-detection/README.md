# Healthcare Threat Detection and Response

## Project Summary

This project focused on improving threat detection and response in a healthcare cybersecurity scenario. The work involved preparing file hashes for analysis, using OSINT methods, defining malicious scoring criteria, and documenting findings.

## Objective

The objective was to analyse file hashes and support threat detection in a healthcare environment where protecting patient data and maintaining service availability are critical.

## Tools Used

- VirusTotal
- OSINT tools
- Hash analysis workflow
- Documentation templates

## Key Activities

- Organised 50 file hashes for investigation.
- Prepared a repeatable workflow for hash analysis.
- Defined criteria for classifying indicators as malicious, suspicious, or benign.
- Reviewed vendor consensus and detection results.
- Documented findings and recommendations.

## Skills Demonstrated

- Threat intelligence enrichment
- Malware indicator analysis
- OSINT investigation
- Healthcare cybersecurity awareness
- Risk-based classification
- Security documentation

## Evidence

This project helped me understand how threat intelligence and hash analysis can support faster decision-making in security operations.
