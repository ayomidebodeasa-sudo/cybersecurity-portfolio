# MuddyWater Threat Intelligence Investigation

## Project Summary

This project focused on investigating MuddyWater, a known threat actor, using MISP and OSINT research to support intelligence-led cyber defence.

## Objective

The objective was to understand how threat intelligence can help SOC teams identify adversary behaviour, organise indicators of compromise, and improve detection and response planning.

## Tools Used

- MISP
- OSINT sources
- MITRE ATT&CK
- Threat intelligence documentation

## Key Activities

- Researched MuddyWater tactics, techniques, and procedures.
- Organised indicators of compromise in MISP.
- Created a structured threat actor profile.
- Mapped behaviours to MITRE ATT&CK.
- Developed defensive recommendations for SOC teams.

## Skills Demonstrated

- Threat actor research
- MISP usage
- IOC management
- MITRE ATT&CK mapping
- Intelligence-led defence
- SOC reporting

## Evidence

Planned evidence to include:

- MISP event screenshots
- IOC table
- MITRE ATT&CK mapping
- Threat actor profile
- Defensive recommendations

## Reflection

This project helped me understand how threat intelligence supports SOC monitoring, detection engineering, and incident response preparation.
